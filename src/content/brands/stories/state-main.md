# State Main

When Canadians think of great dining experiences, State Main is a name that comes up time and again.

State & Main is a Canadian restaurant that takes classic comfort food and gives it a fresh, modern spin.

The restaurant's approach to food combines time-tested recipes with fresh ideas, creating a menu that appeals to a wide range of tastes. State Main understands what Canadian diners want and delivers on that promise consistently.

From coast to coast, State Main remains a trusted choice for Canadians looking for a satisfying meal in a comfortable setting.
